Table Structue

Tables are formatted using the <table> tag. Within a table the structure can be broken down to head, body and footer as seen below. Tables are made of rows, not coloumns, each row using the <tr> (table row) tag and a data field created using the <td> (table data) tag.

<table>

<thead>

<th>Date</th>

<th>Experience</th>

</thead>

<tbody>

<tr>

<td>2015-2020</td>

<td>Creative Director at enchantedbrave.</td>

</tr>

<tr>

<td>2020-2022</td>

<td>CEO of AM&Co.</td>

</tr>

</tbody>

</table>

The above code gives the below output:

Date Experience
2015-2020 Creative Director at enchantedbrave.
2020-2022 CEO of AM&Co.
